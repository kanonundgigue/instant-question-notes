EMアルゴリズムとは？

作成日: 2026-07-23

カテゴリ: statistics

元の質問

EMアルゴリズムとは？

回答

EMアルゴリズム（Expectation-Maximization algorithm、期待値最大化法）は、観測できない潜在変数や欠損データを含む確率モデルのパラメータを、最尤法で推定するための反復アルゴリズムです。直接最大化しにくい観測データの尤度を、潜在変数まで分かっていると仮定した「完全データ」の問題に置き換え、EステップとMステップを交互に実行します。Dempster、Laird、Rubinの1977年の論文が、EステップとMステップを一般形として定式化しました。[1]

何が難しいのか

観測データを \(x\)、観測できない潜在変数を \(z\)、推定したいパラメータを \(\theta\) とします。\(z\) が離散変数なら、求めたいのは観測データ対数尤度

\[ \begin{aligned} \ell(\theta) &= \log p(x \mid \theta) \\ &= \log \sum_z p(x, z \mid \theta) \end{aligned} \]

を最大にする \(\theta\) です。連続な潜在変数の場合は、和の代わりに積分になります。潜在変数について足し合わせた値の対数、つまり「logの中に和がある」形は、微分してもパラメータを解析的に解けないことがあります。一方、\(z\) まで既知とした完全データ対数尤度 \(\log p(x, z \mid \theta)\) は、最大化しやすい形になることがあります。

EステップとMステップ

\(t\) 回目のパラメータを \(\theta^{(t)}\) とすると、EMは次の2段階を繰り返します。[1]

1. Eステップ（Expectation）

\[ Q(\theta \mid \theta^{(t)}) = \mathbb{E}_{z \mid x,\,\theta^{(t)} } \left[ \log p(x, z \mid \theta) \right] \]

現在のパラメータの下で、潜在変数が各状態を取る確率を求め、完全データ対数尤度の条件付き期待値 \(Q\) を作ります。

2. Mステップ（Maximization）

\[ \theta^{(t+1)} = \operatorname*{arg\,max}_{\theta} Q(\theta \mid \theta^{(t)}) \]

Eステップで作った期待完全データ対数尤度を最大にするパラメータへ更新します。

名前から「期待値そのものを最大化する」と読めますが、正確には、Eステップで完全データ対数尤度の条件付き期待値を計算し、Mステップでその関数を最大化する手続きです。Eステップでパラメータを更新するわけでも、Mステップで潜在変数を1つの値に確定するわけでもありません。

混合ガウスモデルで考える

代表例は混合ガウスモデルです。複数の正規分布が混ざって観測値を生成したと考えますが、各観測 \(x_i\) がどの成分から来たかを示す \(z_i\) は観測されません。有限混合分布に対するEMの定式化と性質は、RednerとWalkerによって詳しく整理されています。[2]

  1. 初期化: 各成分 \(k\) の混合比 \(\pi_k\)、平均 \(\mu_k\)、共分散 \(\Sigma_k\) に初期値を置く。
  2. Eステップ: 観測 \(i\) が成分 \(k\) に属する事後確率、責任度 \(r_{ik}\) を計算する。
  3. Mステップ: \(r_{ik}\) を重みとして、各成分の混合比、平均、共分散を再推定する。
  4. 反復: 対数尤度の増加量やパラメータの変化が十分小さくなるまで繰り返す。

Eステップの責任度は、正規密度を \(\mathcal{N}\) と書けば、

\[ r_{ik} = \frac{ \pi_k \mathcal{N}(x_i \mid \mu_k, \Sigma_k) }{ \displaystyle \sum_{j=1}^{K} \pi_j \mathcal{N}(x_i \mid \mu_j, \Sigma_j) } \]

です。0か1かに即断するのではなく、例えば「成分1に0.8、成分2に0.2」のように所属の不確実性を確率のまま保持するのがポイントです。

\(N_k = \sum_{i=1}^{n} r_{ik}\) と置くと、Mステップは次の重み付き推定になります。

  • 混合比を \(\pi_k^{\mathrm{new} } = N_k / n\) で更新する。
  • 平均を \(\mu_k^{\mathrm{new} } = N_k^{-1} \sum_{i=1}^{n} r_{ik} x_i\) で更新する。
  • 共分散を次式で更新する。

\[ \begin{aligned} \Sigma_k^{\mathrm{new} } &= \frac{1}{N_k} \sum_{i=1}^{n} r_{ik} \\ &\quad {}\times (x_i - \mu_k^{\mathrm{new} }) (x_i - \mu_k^{\mathrm{new} })^{\mathsf{T} } \end{aligned} \]

更新された分布を使うと所属確率が変わり、その所属確率で分布を更新すると、またパラメータが変わります。この相互依存を交互更新で解くのがEMです。

なぜ反復するとうまくいくのか

通常のEMでEステップとMステップを正確に実行すると、各反復で観測データの尤度は減少しません。Mステップで改善するのは \(Q\) 関数ですが、その更新が観測データ尤度の非減少も保証するように構成されています。[1] ただし「尤度が下がらない」ことと、「大域的な最大値に到達する」ことは別です。

収束についての注意

EMの尤度列は単調に増え、上に有界ならある値へ収束します。しかし、一般には大域的最適解への収束は保証されず、局所最大値だけでなく鞍点などの停留点へ至る可能性もあります。また、尤度の値が収束しても、追加条件なしにパラメータ列が1点へ収束するとは限りません。Wuはこの違いを整理し、初期値を複数変えて試すことを推奨しています。[3]

  • 初期値依存: 複数の初期値から実行し、最終対数尤度や解の妥当性を比較する。
  • 停止条件: パラメータ変化だけでなく、観測データ対数尤度の変化も確認する。
  • 遅い収束: 解の近くでも更新が小さく、反復回数が多くなる場合がある。
  • 混合モデルの退化: ガウス成分の分散が極端に小さくなるなど、尤度が不健全に大きくなる解へ近づくことがあるため、共分散の正則化や制約が必要になる場合がある。[2]
  • モデル仮定: EMは最適化手順であって、分布の選択や欠損発生メカニズムの仮定が正しいことまでは保証しない。

「欠損値を埋める方法」と考えてよいか

半分は正しく、半分は誤解です。Eステップでは欠けた量に関する条件付き期待値を使いますが、一般には欠損値を1回だけ平均で埋め、その値を確定データとして扱う方法ではありません。潜在変数の条件付き分布に関する完全データ対数尤度の期待値を反復ごとに更新し、その不確実性をパラメータ推定へ反映します。最終目的は、原則として欠損値そのものの確定ではなく、観測データの尤度に基づくパラメータ推定です。

要点

EMアルゴリズムは、潜在変数を現在のモデルで確率的に推定するEステップと、その推定を重みとしてモデルを更新するMステップを交互に行う最尤推定法です。混合分布、欠損データ、因子分析など、完全データなら扱いやすいモデルで力を発揮します。一方で、初期値依存、局所解、収束速度、モデル仮定は別途確認する必要があります。
